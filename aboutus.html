About the Project

Keep Russ Busy was developed as part of Assignment 1 – Look & Feel for ISM 6225: App Development for Analytics. The goal was to design a multi-page website that demonstrates professional web design fundamentals and interactive features using HTML, CSS, and JavaScript.

Design & Technologies

The site consists of four interconnected pages:

  • Home – Overview of the theme and layout
  • About Us – Developer profile and reflection
  • Manage Mishaps – Mock CRUD form to create, read, update, and delete “incidents”
  • Analytics – Visualization using Chart.js showing mishaps by category and severity

Technologies include:

  • HTML5 for structure
  • CSS3 for styling and responsiveness
  • Vanilla JavaScript for interactivity
  • Chart.js for data visualization

Reflection

This project taught me how structure, spacing, and readability influence user experience. It also reinforced my understanding of connecting pages with consistent navigation and styling.

I found it rewarding to see how small pieces of JavaScript could make a simple site interactive and engaging. The Manage Mishaps page demonstrates CRUD logic with no database, while the Analytics page integrates Chart.js for visual insights. Overall, this assignment strengthened both my technical and design confidence as I continue developing applied analytics projects.
